Project: Laravel 10 PHP web application (base skeleton) located at /home/donnie/projects/mr2.
Purpose: server-rendered/API web app using Laravel framework conventions.
Tech stack: PHP 8.1+, Laravel 10, Sanctum, Tinker, PHPUnit 10, Laravel Pint, Vite 5, Axios, Node tooling.
High-level structure:
- app/: application code (controllers, models, services)
- routes/: route definitions (web/api/console/channels)
- resources/: Blade views and frontend assets
- database/: migrations, seeders, factories
- tests/: Unit and Feature tests
- config/: framework and app configuration
- public/: web entrypoint/static assets
- bootstrap/: framework bootstrap
- artisan: Laravel CLI entrypoint
