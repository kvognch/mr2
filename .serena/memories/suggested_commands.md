Core development commands (Linux):
- Install PHP deps: composer install
- Install JS deps: npm install
- Start local app server: php artisan serve
- Start frontend dev server: npm run dev
- Build frontend assets: npm run build
- Run tests: php artisan test
- Alternative test runner: ./vendor/bin/phpunit
- Format PHP code: ./vendor/bin/pint
- Generate app key (first setup): php artisan key:generate
- Run database migrations: php artisan migrate
Useful shell utilities:
- ls, cd, pwd
- rg (ripgrep) for fast search
- find for file discovery
- git status / git diff / git log for VCS
